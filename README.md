# Ajaia Docs

A lightweight collaborative document editor inspired by Google Docs.

## Live Demo

Frontend: [[https://ajaia-docs-ruddy.vercel.app/](https://ajaia-docs-ruddy.vercel.app/)]

Backend: [https://backend-395n.onrender.com]

## Features

- Create documents
- Rename documents
- Rich text editing
- Save and reopen documents
- Upload .txt and .md files
- Share documents with another user
- Persistent storage using SQLite

## Tech Stack

Frontend

- React
- Vite
- Axios
- TipTap

Backend

- Node.js
- Express
- SQLite
